# Cowork 專屬設定

## 沙盒限制

Cowork 在沙盒環境執行，每次 session 結束後設定清除。每次開啟新 session 必須重貼全域設定指令。

## Session 開場載入指令

將以下文字貼到新 Cowork session 開頭：

---

請先讀取以下設定後再開始工作：
https://raw.githubusercontent.com/coldjokenewbie-code/WTF_Under_Construction/main/claude-config/CLAUDE.md

本次任務：[在這裡填入任務描述]

---

確認 Claude 回應「已讀取設定」後再開始工作。

## 模型建議

展演腳本、文件類輕量任務建議切換 Haiku 模型以節省額度。
